# 一、模块名称
M5 AI备课模块 (AI Lesson Preparation Module)

# 二、模块简介
AI备课是LeapMind平台的核心教学准备模块，服务于AI教师即时生成个性化教学内容。它根据学生提问关键词，结合M3薄弱点数据，自动生成结构化教案并编排为PPT课件，供M4即时讲课模块直接使用。

核心定位：AI教师的“即时策略生成器”——围绕学生薄弱点和实时提问，生成查漏补缺式的定制化教学方案。

# 三、主要功能
## 1 AI备课内容生成
输入知识点/章节 + 教学目标 + 年级 + 课时数

AI生成结构化教案：教学目标 → 教学重难点 → 教学过程（导入/新授/练习/总结）→ 板书设计 → 课后作业

支持SSE流式返回，逐段展示生成过程

支持用户修改AI生成的内容（大纲确认、详案编辑）

## 2 薄弱点自适应备课
接收M3传入的薄弱知识点ID及详细掌握数据（掌握度、错误模式、历史错题）

在Prompt中注入教学约束：薄弱环节增加讲解时长、增加变式练习

生成的PPT中，薄弱点对应页面自动添加标记，供M4讲课时特殊处理（放慢语速、加重语气、插入互动提问）

课后作业区分“常规题”与“薄弱补偿题”

## 3 AI动态PPT编排
基础功能：固定结构生成（封面→目标→内容页×N→总结→练习）

高级功能：AI智能决定每页信息密度、图文布局、分页节奏

支持PPT页面增删改、拖拽排序

支持为幻灯片添加备注、互动提问标签

## 4 PPT风格自定义（高级功能2）
提供多套系统预设模板

用户自定义：主色/辅色/背景色/文字色、标题字体/正文字体

风格配置保存为个人模板，一键套用

预览实时生效

## 5 备课内容管理
备课内容列表（标题、科目、课时数、PPT页数、创建时间）

备课内容CRUD（创建、查看、更新、删除）

导出为标准PPTX文件（Apache POI）

“开始讲课”一键跳转M4

## 6 教学风格写入画像（与M6联动）
用户修改PPT风格（配色、字体）→ 写入M6用户画像

用户调整教案结构（增删课时、修改重难点）→ 写入M6

下次生成时自动适配用户偏好

# 四、使用技术

| 层 | 技术 | 语言 | 职责 |
| :--- | :--- | :--- | :--- |
| **前端** | React + TypeScript | TypeScript | PPT渲染组件、风格编辑器（颜色/字体选择器、模板预览）、内容编辑页、流式响应处理 |
| **后端（API层）** | Spring Boot | Java | 备课内容CRUD、PPT模板管理、风格配置管理、PPTX导出（Apache POI） |
| **后端（AI层）** | FastAPI | Python | AI备课内容生成（Prompt链：大纲→详案）、AI动态PPT编排、薄弱点自适应注入 |
| **存储** | MySQL + MinIO/OSS | — | MySQL存备课内容、模板、日志；MinIO存导出的PPTX文件 |
| **缓存/会话** | Redis | — | 流式生成过程中的临时上下文、用户会话 |
| **AI模型** | 通义千问/Qwen-Plus / DeepSeek-V3 | — | 长上下文支持（≥128K），用于教案生成和PPT编排 |
